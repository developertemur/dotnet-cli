# dotnet restore

Terminalga **dotnet restore** kommandasi berish orqali siz dotnet loyihaning \*.csproj faylida yozilgan bog'liqklarni(dependencies) nuget sources(nuget kutubxonalari) dan kompyuteringizga yuklab olasiz.
