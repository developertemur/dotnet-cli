# dotnet build

Terminalga **dontet build** kommandasini yozish orqali siz dotnet loyihani
komilyatsiya qilishingiz mumkin. Ushbu kommanda bajarilishidan oldin 
**dotnet restore** buyrug'ni avtomatik bajaradi.
